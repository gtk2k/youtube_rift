#Youtube Rift
----
Youtubeの動画をOculus Riftで見れるようにするChrome拡張機能。


##操作説明。
インストールすると、Youtubeの動画ページにアクセスしたらアドレスバーの右側にアイコンが表示されます。

###動画プレイヤーがFlashの場合
灰色のアイコンが表示されます。
この場合はOculus Riftの表示はできません。

###動画プレイヤーがHTML5の場合
>黒色のアイコンが表示されます。アイコンをクリックするとOculus Rift用の表示に切り替わり、アイコンが緑色になります。
>再度、アイコンをクリックすると元の表示に戻ります。

###CTRL(control)キー
>CTRLキーでOculus Riftでの表示方法を切り替えることができます。
>(Cover表示⇔Contain表示)

###Zキー
>ZキーでOculus Riftでの表示サイズを段階的に小さくすることができます。

###Xキー
>XキーでOculus Riftでの表示サイズを段階的に大きくすることができます。

###3D動画
>3D動画に対応しています。プレイヤーを"HTML5ステレオビュー"に切り替えてからアイコンをクリックしてください。Oculus Riftで立体で楽しめます。
>3D動画の表示を切り替えた直後は右目用の映像が表示されない場合がありますが、しばらくすると表示されます。(たまにずっと表示されない場合もあります)


###注意
>Oculus Rift表示をしている最中は動画の一時停止やシークなどのコントロールが一切できません。
